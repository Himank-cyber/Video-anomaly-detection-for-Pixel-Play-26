This repository contains a Jupyter Notebook that was made for Pixel Play'26 Video Anomaly Detection Challenge by VLG IITR. 

The primary goal of this project is to:
* Clean the raw dataset
* Build and train a Predictor that can well predict the future frame for normal training data using U-Net architecture
* Add a Generative Adversarial Network module into our framework in light of its success for image generation and video generation
* Build amd train a discriminator network that can identify the abnormal frames generated by the predictor
* Create PSNR scores and save them in CSV file for final submission
